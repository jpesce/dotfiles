# iTerm Configuration Files

## Installation
Install iTerm from the [official website](https://iterm2.com/downloads.html) or via Homebrew:

```sh
brew install --cask iterm2
```

## Settings
Open iTerm, go to `iTerm > Settings` (<kbd>âŒ˜</kbd>+<kbd>,</kbd>). On the tab `General > Settings`,
select 'Load preferences from a custom folder or URL' and point it to the `iterm2`
directory in this repo.
