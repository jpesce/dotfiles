<p align="center">
  <h1 align="center">Raycast</h1>
</p>

Configuration exports for [Raycast](https://www.raycast.com/), a launcher and productivity tool for macOS.

## Installation

```sh
brew install --cask raycast
```

## Files

The `.rayconfig` files are automatic configuration exports (settings, extensions, snippets, etc.) created by Raycast's scheduled export feature (`Settings > Advanced > Import/Export > Configure Export Schedule`).

To restore, use `Settings > Advanced > Import/Export > Import`.
